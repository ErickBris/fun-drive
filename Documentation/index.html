    • Which Eclipse version is needed? - top

    Latest Eclipse version is recommended, which can be downloaded from here:
    https://eclipse.org/downloads/packages/release/Kepler/SR2

    How to open the project in Eclipse? - top

    1. Open Eclipse > Import > Android > "Existing Android Code into Worksplace" > Next
    2. Root Directory > Click Browse button & browse the project folder > Select projects FunDrive,actionbarsherlock,android-saripaar-master,Cropper,google-play-services_lib,Library and SweetAlert
    3. Change Api Above 15- Right Click on project > properties > android > change api level
    4. Then go to 'Project' Menu > Clean.. > Clean All Projects > OK
    5. Wait for few minutes until all errors are gone.

    How to change the package name? - top

    You can change it easily from Eclipse. Here are the steps:

    • Right Cilck 'FunDrive' project from Package Explorer > Android Tools > Rename Application Package > Finish.
    • Refactroing dailog present and click Finish
    • Package Name Changed in AndroidMainfest.xml but not in Application Package hierarchy so select Package name Right Click > Go to Refactor > Rename >Enter Package Name > Finish

    How to change app name? - top

    1. Open Eclipse >FunDrive> res > values > strings.xml
    2. Select 'strings.xml' tab next to 'Resources' tab
    3. Enter your app name inside "app_name" string tag:
      <string name="app_name">FunDrive</string>
    4. if you want change Project name also then >Right Cilck 'FunDrive' project from Package Explorer >Refactor > Rename >Enter Application Name > Finish
    This will change both title inside the app & app icon name.

    How to Configure Webservice In Your Server? - top

    Local set up

    1. First of all find the FunDrive folder from the downloaded package.
    2. Copy these folder and paste it in the htdocs folder(if you have Wampp installed then put it into www)
    3. Then create a database.
    4. That’s all,now run the web service and follow the steps: http://localhost/FunDrive/index
    Live set up

    1. First of all find the FunDrive folder from the downloaded package.
    2. Upload this into your live server
    3. Then create a database.
    4. That’s all,now run the web service follow the steps: http://mydomain.com/FunDrive/index
    Login Details

    Username: fundrive   Password: fundrive

    How to change AdMob publisher id? - top

    1. Open Eclipse > FunDrive > res > values > strings.xml
    2. Select 'strings.xml' tab next to 'Resources' tab
    3. Enter your AdMob publisher id inside "admob_publisher_id" string tag:
      <string name="admob_publisher_id">Your AdMob ID Here</string>

    How to change StartApp Ad Id/Developer Id? - top

    1. Open Eclipse >FunDrive> res > values > strings.xml
    2. Select 'strings.xml' tab next to 'Resources' tab
    3. Enter your StartApp Application id inside "startapp_app_id" string tag:
      <string name="startapp_app_id">Your StartApp Application ID</string>
    4. Enter your StartApp Developer id inside "startapp_dev_id" string tag:
      <string name="startapp_dev_id">Your StartApp Developer ID</string>
    5. Open AndroidMainfiest.xml > Change in Startapp Code

    How to Change Google Analytics Tracking id in Application ? - top

    1. Open Eclipse > FunDrive > res > values > analytics.xml
    2. Select 'analytics.xml' tab next to 'Resources' tab
    3. Enter your google tracking id inside "ga_trackingId" string tag:
      <string name="ga_trackingId">Your Google Tracking ID Here</string>
